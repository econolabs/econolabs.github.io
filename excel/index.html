1. Базовые функции генерации случайных чисел

СЛЧИС() — возвращает случайное дробное число от 0 до 1 (включительно). Функция не имеет аргументов и пересчитывается при каждом изменении листа.

СЛУЧМЕЖДУ(нижняя_граница; верхняя_граница) — возвращает случайное целое число в указанном диапазоне. Например, =СЛУЧМЕЖДУ(1; 100) вернет целое число от 1 до 100.

3. Как остановить пересчет случайных чисел

По умолчанию функции СЛЧИС() и СЛУЧМЕЖДУ() пересчитываются при каждом изменении листа. Чтобы "заморозить" значения:

  1. Выделите ячейки с формулами
  2. Скопируйте их (Ctrl+C)
  3. Выполните: Специальная вставка → Значения (Ctrl+Shift+V или через правую кнопку мыши)

Альтернативно: можно отключить автоматический пересчет в параметрах формул (Файл → Параметры → Формулы → Вычисления вручную).

4. Новая функция Excel 365: МАССИВСЛЧИС()

В новых версиях Excel 365 появилась мощная функция:

=МАССИВСЛЧИС(строки; [столбцы]; [минимальное]; [максимальное]; [целое_число])

Примеры:

  • =МАССИВСЛЧИС(5; 3; 1; 100; ИСТИНА) — матрица 5×3 со случайными целыми числами 1-100
  • =МАССИВСЛЧИС(10; 1; 0; 1) — 10 случайных дробных чисел 0-1

5. Пакет анализа для сложных распределений

Для генерации случайных чисел по заданным статистическим распределениям (нормальное, биномиальное и др.):

  1. Включите надстройку Пакет анализа (Файл → Параметры → Надстройки → Перейти → Пакет анализа)
  2. Перейдите: Данные → Анализ данных → Генерация случайных чисел
  3. Выберите тип распределения и параметры

6. Практические примеры применения

Создание тестовых данных

Генерация случайных цен, количеств, ID для тестирования отчетов и формул.

=СЛУЧМЕЖДУ(100; 10000) — случайная цена
=СЛУЧМЕЖДУ(1; 100) — случайное количество
Розыгрыш призов/жеребьевка

Выбор случайного победителя из списка участников.

=ИНДЕКС(A2:A100; СЛУЧМЕЖДУ(1; СЧЁТЗ(A2:A100)))
Важно знать:
  • Excel использует псевдослучайные числа, которые подходят для большинства практических задач (моделирование, тестирование, выборки), но не для криптографии
  • В русской версии Excel десятичный разделитель — запятая (0,743562), а разделитель аргументов — точка с запятой (;)
  • Для воспроизводимости случайных последовательностей можно задать начальное значение через VBA

  • Функции для проверки гипотез и корреляции

1. Основные описательные статистики

Среднее арифметическое

=СРЗНАЧ(диапазон)

Центральная тенденция данных. Чувствительно к выбросам.

Пример: =СРЗНАЧ(A2:A100)
Медиана

=МЕДИАНА(диапазон)

Значение, которое делит данные пополам. Устойчиво к выбросам.

50-й процентиль
Мода

=МОДА(диапазон)

Наиболее часто встречающееся значение.

=МОДА.ОД(A2:A100) — одна мода =МОДА.НСК(A2:A100) — несколько мод
Сумма

=СУММ(диапазон)

Общая сумма всех значений.

=СУММ(A2:A100) или =СУММПРОИЗВ()
Пример анализа случайных чисел:

Сгенерируем 100 случайных чисел от 1 до 100 и проанализируем:

Случайные числа: =СЛУЧМЕЖДУ(1;100) в A2:A101
Среднее: =СРЗНАЧ(A2:A101) → ~50.5
Медиана: =МЕДИАНА(A2:A101) → ~50
Мода: =МОДА.ОД(A2:A101) → зависит от выборки

2. Меры изменчивости данных

Стандартное отклонение

=СТАНДОТКЛОН.В(диапазон)

Для всей генеральной совокупности.

=СТАНДОТКЛОН.Г() — для выборки
Дисперсия

=ДИСП.В(диапазон)

Квадрат стандартного отклонения.

=ДИСП.Г() — для выборки
Размах

=МАКС(диапазон)-МИН(диапазон)

Разность между максимальным и минимальным значениями.

Квартили

=КВАРТИЛЬ.ВКЛ(диапазон; квартиль)

Квартиль = 1 (25%), 2 (50% = медиана), 3 (75%).

=КВАРТИЛЬ.ИСКЛ() — другой метод
Процентили

=ПРОЦЕНТИЛЬ.ВКЛ(диапазон; k)

k от 0 до 1. Например, 0.9 = 90-й процентиль.

Межквартильный размах

=Q3 - Q1

Разность между 75-м и 25-м процентилями.

=КВАРТИЛЬ.ВКЛ(A:A;3)-КВАРТИЛЬ.ВКЛ(A:A;1)
Пример анализа изменчивости:
Стандартное отклонение: =СТАНДОТКЛОН.В(A2:A101) → ~28.9
Дисперсия: =ДИСП.В(A2:A101) → ~835
Размах: =МАКС(A2:A101)-МИН(A2:A101) → ~99
Q1 (25%): =КВАРТИЛЬ.ВКЛ(A2:A101;1) → ~25
Q3 (75%): =КВАРТИЛЬ.ВКЛ(A2:A101;3) → ~75

3. Анализ формы распределения

Асимметрия (Skewness)

=СКОС(диапазон)

Мера симметрии распределения:

  • > 0 — правосторонняя асимметрия
  • = 0 — симметричное распределение
  • < 0 — левосторонняя асимметрия
Эксцесс (Kurtosis)

=ЭКСЦЕСС(диапазон)

Мера "остроты" распределения:

  • > 0 — островершинное (пик)
  • = 0 — нормальное распределение
  • < 0 — пологое распределение
Проверка на нормальность

=НОРМ.РАСП(x; среднее; ст_откл; ИСТИНА)

Вероятность в нормальном распределении.

Для критериев: =НОРМ.ОБР(вероятность; среднее; ст_откл)
Стандартизация Z-оценка

=СТАНДОТКЛОН.ОТКЛ(x; среднее; ст_откл)

Преобразует значение в стандартную Z-оценку.

=СТАНДОТКЛОН.ОКРУГЛ() — устаревшая версия
Пример анализа распределения случайных чисел:
Асимметрия: =СКОС(A2:A101) → близко к 0 (равномерное распределение)
Эксцесс: =ЭКСЦЕСС(A2:A101) → около -1.2 (равномерное)
Z-оценка для значения 75: 
=СТАНДОТКЛОН.ОТКЛ(75; СРЗНАЧ(A2:A101); СТАНДОТКЛОН.В(A2:A101))
→ ~0.85 (на 0.85 стандартных отклонений выше среднего)

=PEARSON() — английский вариант

=КОВАРИАЦИЯ.В() — для выборки
Критерии Стьюдента

=СТЬЮДЕНТ.РАСП(x; степени_свободы; хвосты)

t-распределение Стьюдента.

хвосты: 1 — односторонний, 2 — двусторонний
Хи-квадрат распределение

=ХИ2.РАСП(x; степени_свободы; интегральная)

Для проверки гипотез о распределении.

Подсчет элементов

=СЧЁТ(диапазон)

Количество чисел в диапазоне.

=СЧЁТЕСЛИ(диапазон; условие)

Количество ячеек, удовлетворяющих условию.

=СЧЁТЕСЛИМН() — несколько условий

5. Практическое применение: анализ сгенерированных данных

1. Генерация тестовых данных

Создаем 1000 значений с нормальным распределением:

=НОРМ.ОБР(СЛЧИС(); 100; 15)

Среднее = 100, SD = 15

2. Анализ распределения

Проверяем соответствие нормальному распределению:

  • СКОС ≈ 0
  • ЭКСЦЕСС ≈ 0
  • 68% данных в ±1SD
3. Выявление выбросов

Находим значения за пределами 3σ:

=СЧЁТЕСЛИ(A:A;"<"&D2-3*D3)
+СЧЁТЕСЛИ(A:A;">"&D2+3*D3)

D2=среднее, D3=стандартное отклонение

Советы по использованию:
  • Используйте именованные диапазоны для удобства: =СРЗНАЧ(Данные)
  • Комбинируйте функции: =СТАНДОТКЛОН.В(ЕСЛИ(Условие; Данные))
  • Для динамических диапазонов: =СРЗНАЧ(A2:ИНДЕКС(A:A; СЧЁТ(A:A)))
  • Используйте Таблицы Excel для автоматического расширения формул
  • Для визуализации применяйте гистограммы и ящики с усами
Важно помнить:
  • Функции с .В — для всей совокупности, .Г — для выборки
  • Для проверки гипотез используйте уровень значимости α = 0.05
  • Нормальное распределение: 68% в ±1σ, 95% в ±2σ, 99.7% в ±3σ
  • Все формулы работают в Excel 2010 и выше

Визуализация распределения случайных данных в Excel

Графики и диаграммы для анализа распределения

Как визуализировать и анализировать сгенерированные случайные числа

Визуальные методы анализа:
  • Гистограммы для распределения частот
  • Ящики с усами (box plot) для статистик
  • Графики плотности и нормального распределения
  • QQ-plot для проверки нормальности
  • Точечные диаграммы и скрипичные графики

1. Гистограмма распределения

Создание гистограммы
  1. Выделите данные
  2. Вставка → Диаграммы → Гистограмма
  3. Настройте интервалы корзин
Автоматические интервалы
=ЧАСТОТА(данные; карманы)

Для создания таблицы частот вручную

Пример для случайных чисел 1-100: 
1. Создаем интервалы (карманы):
10, 20, 30, ..., 100 в B2:B11

2. Формула частот в C2:C11:
=ЧАСТОТА(A2:A101; B2:B11)

3. Строим гистограмму по C2:C11
Равномерное распределение → плоская гистограмма
Нормальное распределение → колокол

2. Ящик с усами (Box Plot)

  1. Выделите данные

Анализ по box plot:
• Короткий ящик → малая дисперсия
• Сдвиг медианы → асимметрия
• Длинные усы → выбросы

3. Графики проверки нормальности

График нормального распределения

Сравнение с теоретической нормальной кривой:

  1. Среднее: =СРЗНАЧ(A:A)
  2. Ст. отклонение: =СТАНДОТКЛОН.В(A:A)
  3. Нормальная кривая:
    =НОРМ.РАСП(x; среднее; ст_откл; ЛОЖЬ)
  4. Строим два ряда: данные и кривую
QQ-plot (квантиль-квантиль)

Сравнение квантилей данных с нормальными:

  1. Сортируем данные по возрастанию
  2. Вычисляем теоретические квантили:
    =НОРМ.ОБР((i-0.5)/n; среднее; ст_откл)
  3. Строим точечную диаграмму
  4. Прямая линия → нормальное распределение
Пример QQ-plot для 100 случайных чисел: 
Данные в A2:A101
Ранг в B2:B101:
=СТРОКА(A2)-1

Вероятность в C2:C101:
=(B2-0.5)/100

Теоретич. квантиль в D2:D101:
=НОРМ.ОБР(C2; СРЗНАЧ($A$2:$A$101); 
СТАНДОТКЛОН.В($A$2:$A$101))
Интерпретация:
• Точки на прямой → нормальное
• S-образная кривая → тяжелые хвосты
• Выгнутость → асимметрия
• Выбросы → отклонения от линии

4. Другие типы визуализации

Скрипичный график (Violin Plot)

Комбинация box plot и плотности:

  1. Гистограмма с зеркальным отражением
  2. Показывает форму распределения
  3. Ширина = плотность вероятности
  4. Добавляем box plot поверх
График плотности (KDE)

Сглаженная оценка плотности:

≈ ГИСТОГРАММА + сглаживание

Создание в Excel:

  1. Рассчитать плотность ядра
  2. Использовать сглаженную линию
  3. Или добавить в Power BI
Накопительная гистограмма

Кумулятивное распределение:

  1. Частоты: =ЧАСТОТА()
  2. Накопленные: =СУММ($C$2:C2)
  3. Доля: =D2/СУММ($C$2:$C$11)
  4. Строим график с накоплением
