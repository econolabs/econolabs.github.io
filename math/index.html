Множества, операции над множествами. Диаграммы Эйлера–Венна

Множество - совокупность объектов, объединенных общим свойством. Обозначается заглавными буквами: A, B, C...

Основные операции над множествами:
Операция Обозначение Определение Пример
Объединение A ∪ B Множество, содержащее все элементы, которые принадлежат хотя бы одному из множеств A или B A = {1,2,3}, B = {3,4,5}
A ∪ B = {1,2,3,4,5}
Пересечение A ∩ B Множество, содержащее все элементы, которые принадлежат и A, и B одновременно A = {1,2,3}, B = {3,4,5}
A ∩ B = {3}
Разность A \ B Множество, содержащее элементы, которые принадлежат A, но не принадлежат B A = {1,2,3}, B = {3,4,5}
A \ B = {1,2}
Дополнение Ac или Ā Множество всех элементов универсума, не принадлежащих A U = {1,2,3,4,5}, A = {1,2}
Ac = {3,4,5}
Свойства операций над множествами:
  • A ∪ B = B ∪ A (коммутативность объединения)
  • A ∩ B = B ∩ A (коммутативность пересечения)
  • A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C) (дистрибутивность)
  • A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C) (дистрибутивность)

Диаграммы Эйлера-Венна используются для наглядного представления множеств и операций между ними. Множества изображаются в виде кругов или других фигур, а их взаимное расположение показывает отношения между ними.

Рациональные числа. Обыкновенные и десятичные дроби, проценты, бесконечные периодические дроби

Рациональные числа - числа, которые можно представить в виде дроби m/n, где m - целое число, n - натуральное число. Множество рациональных чисел обозначается ℚ.

Основные формы представления рациональных чисел:
1. Обыкновенные дроби:

Записываются в виде a/b, где a - числитель, b - знаменатель.

Примеры:

  • 1/2 = 0,5 (правильная дробь)
  • 5/3 = 1⅔ (неправильная дробь)
  • -3/4 = -0,75 (отрицательная дробь)
  • 7/1 = 7 (целое число как дробь)
2. Десятичные дроби:

Конечные десятичные дроби:

  • 1/2 = 0,5
  • 3/4 = 0,75
  • 7/8 = 0,875

Бесконечные периодические дроби:

  • 1/3 = 0,333... = 0,(3)
  • 2/7 = 0,285714285714... = 0,(285714)
  • 5/6 = 0,8333... = 0,8(3) (смешанная периодическая)
3. Проценты:

Один процент (1%) - это сотая часть числа: 1% = 1/100 = 0,01

Примеры преобразований:

  • 25% = 25/100 = 1/4 = 0,25
  • 150% = 150/100 = 3/2 = 1,5
  • 0,5% = 0,5/100 = 0,005

Арифметические операции с рациональными числами, преобразования числовых выражений

Арифметические операции - основные математические действия над числами: сложение, вычитание, умножение и деление.

Операции с обыкновенными дробями:
1. Сложение и вычитание дробей:

При одинаковых знаменателях: a/c ± b/c = (a ± b)/c

При разных знаменателях: приводим к общему знаменателю

Примеры:

  • 1/3 + 2/3 = (1+2)/3 = 3/3 = 1
  • 1/2 + 1/3 = 3/6 + 2/6 = 5/6
  • 5/8 - 1/4 = 5/8 - 2/8 = 3/8
2. Умножение дробей:

a/b × c/d = (a × c)/(b × d)

Примеры:

  • 2/3 × 3/4 = (2×3)/(3×4) = 6/12 = 1/2
  • 5/7 × 2/5 = (5×2)/(7×5) = 10/35 = 2/7
  • -3/4 × 2/3 = -6/12 = -1/2
3. Деление дробей:

a/b ÷ c/d = a/b × d/c = (a × d)/(b × c)

Примеры:

  • 2/3 ÷ 3/4 = 2/3 × 4/3 = 8/9
  • 5/6 ÷ 2/3 = 5/6 × 3/2 = 15/12 = 5/4
  • -4/5 ÷ 2/7 = -4/5 × 7/2 = -28/10 = -14/5
Свойства арифметических операций:
Свойство Формулировка Пример
Коммутативность сложения a + b = b + a 2/3 + 1/4 = 1/4 + 2/3
Ассоциативность сложения (a + b) + c = a + (b + c) (1/2 + 1/3) + 1/6 = 1/2 + (1/3 + 1/6)
Коммутативность умножения a × b = b × a 3/4 × 2/5 = 2/5 × 3/4
Ассоциативность умножения (a × b) × c = a × (b × c) (2/3 × 3/4) × 4/5 = 2/3 × (3/4 × 4/5)
Дистрибутивность a × (b + c) = a × b + a × c 2 × (1/3 + 1/4) = 2×1/3 + 2×1/4
Преобразования числовых выражений:
1. Раскрытие скобок:

С положительным знаком: a + (b + c) = a + b + c

С отрицательным знаком: a - (b + c) = a - b - c

Примеры:

  • 5 + (3 - 2) = 5 + 3 - 2 = 6
  • 7 - (4 + 1) = 7 - 4 - 1 = 2
  • 2/3 + (1/4 - 1/6) = 2/3 + 1/4 - 1/6
2. Приведение подобных слагаемых:

ax + bx = (a + b)x

Примеры:

  • 3x + 5x = 8x
  • 2a + 3b + 4a - b = 6a + 2b
  • 1/2x + 1/3x = 3/6x + 2/6x = 5/6x
3. Упрощение сложных выражений:

Пример 1: (2/3 + 1/4) × 12

= 2/3 × 12 + 1/4 × 12 = 8 + 3 = 11

Пример 2: 5/6 ÷ (2/3 - 1/2)

= 5/6 ÷ (4/6 - 3/6) = 5/6 ÷ 1/6 = 5/6 × 6/1 = 5

Пример 3: (1/2 + 1/3) × (1/4 - 1/6)

= (3/6 + 2/6) × (3/12 - 2/12) = 5/6 × 1/12 = 5/72

Порядок выполнения операций:
  1. Выражения в скобках
  2. Умножение и деление (слева направо)
  3. Сложение и вычитание (слева направо)

Пример: 2 + 3 × (4 - 1) ÷ 3 = 2 + 3 × 3 ÷ 3 = 2 + 9 ÷ 3 = 2 + 3 = 5

Применение дробей и процентов для решения прикладных задач

Дроби и проценты широко используются в реальной жизни: финансовые расчеты (проценты по кредитам, вкладам), статистика, кулинария, строительство, медицина и другие области.

Действительные числа. Рациональные и иррациональные числа

Действительные числа включают все рациональные и иррациональные числа. Иррациональные числа нельзя представить в виде дроби m/n (например, √2, π, e).

Арифметические операции с действительными числами

Операции с действительными числами сохраняют все свойства операций с рациональными числами. Особенности: при работе с иррациональными числами часто используются приближенные значения.

Приближённые вычисления, правила округления, прикидка и оценка результата вычислений

Правила округления: если следующая цифра меньше 5, округляем в меньшую сторону; если 5 или больше - в большую. Прикидка позволяет быстро оценить порядок результата. Оценка погрешности важна для понимания точности вычислений.

Тождества и тождественные преобразования. Уравнение, корень уравнения

Тождества и тождественные преобразования:
Определение тождества:

Тождество - равенство, которое верно при всех допустимых значениях входящих в него переменных.

Обозначение: ≡ (знак тождественного равенства)

Примеры тождеств:

  • (a + b)² ≡ a² + 2ab + b²
  • a² - b² ≡ (a - b)(a + b)
  • sin²α + cos²α ≡ 1
  • a + b ≡ b + a (коммутативность сложения)
Тождественные преобразования:

Преобразования, не изменяющие значения выражения при любых допустимых значениях переменных.

Основные виды тождественных преобразований:

  • Раскрытие скобок: a(b + c) → ab + ac
  • Приведение подобных: 2x + 3x → 5x
  • Разложение на множители: a² - b² → (a - b)(a + b)
  • Сокращение дробей: (ab)/(ac) → b/c
Примеры тождественных преобразований:

Пример 1: Упростить выражение 3(x + 2) - 2(x - 1)

3x + 6 - 2x + 2 = x + 8

Пример 2: Доказать тождество (a - b)² ≡ a² - 2ab + b²

(a - b)² = (a - b)(a - b) = a² - ab - ab + b² = a² - 2ab + b²

Пример 3: Упростить дробь (x² - 4)/(x - 2)

(x² - 4)/(x - 2) = ((x - 2)(x + 2))/(x - 2) = x + 2 (при x ≠ 2)

Уравнения и корни уравнений:
Определение уравнения:

Уравнение - равенство, содержащее переменную (неизвестное), которое требуется решить.

Общий вид: f(x) = g(x)

Примеры уравнений:

  • 2x + 3 = 7 (линейное)
  • x² - 5x + 6 = 0 (квадратное)
  • sin x = 0.5 (тригонометрическое)
Корень уравнения:

Корень уравнения - значение переменной, которое обращает уравнение в верное равенство.

Обозначение: Если x₀ - корень уравнения f(x) = g(x), то f(x₀) = g(x₀)

Примеры нахождения корней:

  • Уравнение: 2x - 4 = 0
    Корень: x = 2, так как 2×2 - 4 = 0
  • Уравнение: x² - 4 = 0
    Корни: x₁ = 2, x₂ = -2, так как 2² - 4 = 0 и (-2)² - 4 = 0
Классификация уравнений:
Тип уравнения Общий вид Пример Количество корней
Линейное ax + b = 0 3x - 6 = 0 1 корень (если a ≠ 0)
Квадратное ax² + bx + c = 0 x² - 5x + 6 = 0 0, 1 или 2 корня
Дробно-рациональное P(x)/Q(x) = 0 (x-1)/(x+2) = 0 Корни P(x)=0, кроме корней Q(x)
Иррациональное √f(x) = g(x) √(x+1) = x-1 Требуется проверка
Равносильные уравнения:

Определение: Уравнения называются равносильными, если они имеют одинаковые корни.

Пример: Уравнения 2x = 6 и x = 3 равносильны

Основные преобразования, сохраняющие равносильность:

  • Прибавление одного и того же числа к обеим частям
  • Умножение обеих частей на одно и то же ненулевое число
  • Тождественные преобразования одной или обеих частей
Алгоритм решения уравнений:
  1. Определить ОДЗ (область допустимых значений)
  2. Выполнить тождественные преобразования
  3. Привести уравнение к простейшему виду
  4. Найти корни
  5. Проверить корни на принадлежность ОДЗ
  6. Записать ответ

Пример: Решить уравнение (x-2)/(x+1) = 0

ОДЗ: x ≠ -1
Числитель равен нулю: x - 2 = 0 → x = 2
Проверка ОДЗ: 2 ≠ -1 - верно
Ответ: x = 2

Неравенство, решение неравенства. Метод интервалов. Решение целых и дробно-рациональных уравнений и неравенств

Основные понятия о неравенствах:
Определение неравенства:

Неравенство - соотношение между числами или выражениями, показывающее, что одно число больше или меньше другого.

Виды неравенств:

  • a > b (a больше b)
  • a < b (a меньше b)
  • a ≥ b (a больше или равно b)
  • a ≤ b (a меньше или равно b)
Свойства неравенств:
  • Если a > b, то a + c > b + c
  • Если a > b и c > 0, то ac > bc
  • Если a > b и c < 0 , то ac < bc
  • Если a > b и b > c, то a > c (транзитивность)
Метод интервалов:
Алгоритм метода интервалов:
  1. Найти нули функции (решить уравнение f(x) = 0)
  2. Найти точки разрыва (где функция не определена)
  3. Отметить найденные точки на числовой прямой
  4. Определить знак функции на каждом интервале
  5. Выбрать интервалы, удовлетворяющие неравенству
  6. Записать ответ с учетом строгости неравенства
Пример 1: Решение целого неравенства

Решить неравенство: (x - 2)(x + 3) > 0

Решение:

  1. Нули функции: x = 2, x = -3
  2. Точек разрыва нет
  3. Отмечаем точки на числовой прямой:

---○-------○---→

-3        2

4. Определяем знаки:

  • При x < -3: (-) × (-) = +
  • При -3 < x < 2: (-) × (+) = -
  • При x > 2: (+) × (+) = +

5. Выбираем интервалы со знаком + (т.к. неравенство > 0)

Ответ: x ∈ (-∞; -3) ∪ (2; +∞)

Пример 2: Решение дробно-рационального неравенства

Решить неравенство: (x - 1)/(x + 2) ≤ 0

Решение:

  1. Нуль числителя: x = 1
  2. Точка разрыва (нуль знаменателя): x = -2
  3. Отмечаем точки на числовой прямой (○ - нуль, ● - точка разрыва):

---●-------○---→

-2        1

4. Определяем знаки:

  • При x < -2: (-)/(-) = +
  • При -2 < x < 1: (-)/(+) = -
  • При x > 1: (+)/(+) = +

5. Выбираем интервалы со знаком - (т.к. неравенство ≤ 0), включаем точку x = 1

Ответ: x ∈ (-2; 1]

Решение различных типов неравенств:
Линейные неравенства:

Пример: 3x - 6 > 0

3x > 6 → x > 2

Ответ: (2; +∞)

Квадратные неравенства:

Пример: x² - 5x + 6 < 0

Разложим на множители: (x - 2)(x - 3) < 0

Методом интервалов получаем: x ∈ (2; 3)

Дробно-рациональные неравенства:

Пример: (x² - 4)/(x - 1) ≥ 0

Разложим числитель: ((x - 2)(x + 2))/(x - 1) ≥ 0

Критические точки: x = -2, x = 1, x = 2

Методом интервалов: x ∈ [-2; 1) ∪ [2; +∞)

Особые случаи:
Тип неравенства Особенности решения Пример
Строгое неравенство Точки не включаются в ответ (x-1)/(x+2) < 0 → x ∈ (-2; 1)
Нестрогое неравенство Нули числителя включаются, нули знаменателя - нет (x-1)/(x+2) ≤ 0 → x ∈ (-2; 1]
Неравенство с кратными корнями Знак не меняется при переходе через корень четной кратности (x-1)²(x+2) > 0
Проверка решения:

Всегда проверяйте:

  • Принадлежность точек ОДЗ
  • Правильность включения/исключения граничных точек
  • Знак на каждом интервале

Пример проверки: Для x ∈ (-2; 1] возьмем:

  • x = 0 → (0-1)/(0+2) = -0.5 < 0 - верно
  • x = 1 → (1-1)/(1+2) = 0 ≤ 0 - верно
  • x = -1 → (-1-1)/(-1+2) = -2 < 0 - верно

Степень с целым показателем. Стандартная форма записи действительного числа

Степень с целым показателем:
Определение степени:

Степень числа a с натуральным показателем n - произведение n множителей, каждый из которых равен a:

aⁿ = a × a × a × ... × a (n раз)

Примеры:

  • 2³ = 2 × 2 × 2 = 8
  • 5² = 5 × 5 = 25
  • (-3)³ = (-3) × (-3) × (-3) = -27
Степень с нулевым и отрицательным показателем:

Нулевая степень: a⁰ = 1 (при a ≠ 0)

Отрицательная степень: a⁻ⁿ = 1/aⁿ (при a ≠ 0)

Примеры:

  • 7⁰ = 1
  • 2⁻³ = 1/2³ = 1/8 = 0,125
  • (-2)⁻² = 1/(-2)² = 1/4 = 0,25

Арифметический корень натуральной степени. Свойства арифметического корня. Действия с арифметическими корнями n–ой степени

Арифметический корень натуральной степени:
Определение:

Арифметический корень n-й степени из неотрицательного числа a - это такое неотрицательное число b, что bⁿ = a.

Обозначение: √[n]{a} = b, где:

  • n - показатель корня (натуральное число, n ≥ 2)
  • a - подкоренное выражение (a ≥ 0)
  • b - значение корня (b ≥ 0)

Примеры:

  • √[2]{9} = 3, так как 3² = 9
  • √[3]{8} = 2, так как 2³ = 8
  • √[4]{16} = 2, так как 2⁴ = 16
  • √[2]{0} = 0, так как 0² = 0
Особые случаи:

Квадратный корень: √[2]{a} = √a (n = 2)

Кубический корень: √[3]{a} (n = 3)

Корень из нуля: √[n]{0} = 0 для любого n

Корень из единицы: √[n]{1} = 1 для любого n

Свойства арифметических корней:
Свойство Формула Ограничения Пример
Корень из произведения √[n]{ab} = √[n]{a} · √[n]{b} a ≥ 0, b ≥ 0 √[3]{8·27} = √[3]{8}·√[3]{27} = 2·3 = 6
Корень из частного √[n]{a/b} = √[n]{a} / √[n]{b} a ≥ 0, b > 0 √[2]{9/4} = √9/√4 = 3/2
Корень из степени √[n]{aᵐ} = (√[n]{a})ᵐ a ≥ 0 √[3]{8²} = (√[3]{8})² = 2² = 4
Степень корня (√[n]{a})ᵐ = √[n]{aᵐ} a ≥ 0 (√[2]{9})³ = √[2]{9³} = √729 = 27
Корень из корня √[m]{√[n]{a}} = √[mn]{a} a ≥ 0 √[2]{√[3]{64}} = √[6]{64} = 2
Основное свойство (√[n]{a})ⁿ = a a ≥ 0 (√[3]{8})³ = 2³ = 8
Примеры применения свойств:

Пример 1: Упростить √[3]{54} + √[3]{2}

√[3]{54} = √[3]{27·2} = √[3]{27}·√[3]{2} = 3√[3]{2}
3√[3]{2} + √[3]{2} = 4√[3]{2}

Пример 2: Упростить √[4]{81x⁸y¹²}

√[4]{81}·√[4]{x⁸}·√[4]{y¹²} = 3·x²·y³ = 3x²y³

Пример 3: Вычислить √[3]{√{64}}

√[3]{√{64}} = √[3]{8} = 2

Действия с арифметическими корнями:
Сложение и вычитание корней:

Корни можно складывать и вычитать только при одинаковых показателях и подкоренных выражениях.

Примеры:

  • 3√2 + 5√2 = 8√2
  • 7√[3]{5} - 2√[3]{5} = 5√[3]{5}
  • √2 + √3 - нельзя упростить
  • √2 + √8 = √2 + 2√2 = 3√2 (т.к. √8 = 2√2)
Умножение корней:

При одинаковых показателях: √[n]{a} · √[n]{b} = √[n]{ab}

При разных показателях: приводим к общему показателю

Примеры:

  • √2 · √8 = √16 = 4
  • √[3]{4} · √[3]{2} = √[3]{8} = 2
  • √2 · √[3]{3} = 2^{1/2} · 3^{1/3} = 2^{3/6} · 3^{2/6} = √[6]{2³·3²} = √[6]{72}
Деление корней:

При одинаковых показателях: √[n]{a} ÷ √[n]{b} = √[n]{a/b}

Примеры:

  • √18 ÷ √2 = √{18/2} = √9 = 3
  • √[3]{54} ÷ √[3]{2} = √[3]{54/2} = √[3]{27} = 3
  • (5√6) ÷ (2√3) = 5/2 · √{6/3} = 2,5√2
Возведение корня в степень:

(√[n]{a})ᵐ = √[n]{aᵐ}

Примеры:

  • (√5)² = 5
  • (√[3]{2})⁶ = √[3]{2⁶} = √[3]{64} = 4
  • (2√3)² = 4·3 = 12
Освобождение от иррациональности в знаменателе:
Методы устранения иррациональности:

Случай 1: В знаменателе один корень

a/√b = (a√b)/b

Пример: 3/√5 = (3√5)/5

Случай 2: В знаменателе сумма/разность корней

Используем формулу сопряженного выражения

Пример: 1/(√a + √b) = (√a - √b)/(a - b)

Пример: 2/(√3 - 1) = 2(√3 + 1)/(3 - 1) = (2√3 + 2)/2 = √3 + 1

Практические примеры вычислений:

Пример 1: Вычислить √12 - √27 + √48

√12 = 2√3, √27 = 3√3, √48 = 4√3
2√3 - 3√3 + 4√3 = 3√3

Пример 2: Упростить (√6 + √2)²

(√6 + √2)² = 6 + 2√12 + 2 = 8 + 4√3

Пример 3: Освободиться от иррациональности: 5/(√7 - √2)

5/(√7 - √2) = 5(√7 + √2)/(7 - 2) = 5(√7 + √2)/5 = √7 + √2

Важные замечания:
  • Арифметический корень всегда неотрицателен
  • Подкоренное выражение неотрицательно для корней четной степени
  • При преобразованиях проверяйте ОДЗ
  • Корни нечетной степени определены для отрицательных чисел, но это уже не арифметические корни

Решение иррациональных уравнений и неравенств

Иррациональные уравнения:
Определение:

Иррациональное уравнение - уравнение, в котором переменная содержится под знаком корня (радикала).

Общий вид: √[n]{f(x)} = g(x), где n ≥ 2

Примеры иррациональных уравнений:

  • √(x + 2) = 3
  • √[3]{2x - 1} = x - 1
  • √(x + 1) + √(x - 2) = 5
  • √(x² - 4) = x - 2
Основные методы решения иррациональных уравнений:
Метод 1: Возведение в степень

Используется для уравнений вида √[n]{f(x)} = g(x)

Алгоритм:

  1. Найти ОДЗ: f(x) ≥ 0 (для четных n)
  2. Возвести обе части в степень n
  3. Решить полученное уравнение
  4. Проверить корни в исходном уравнении

Пример 1: √(x + 2) = 3

ОДЗ: x + 2 ≥ 0 → x ≥ -2
Возводим в квадрат: x + 2 = 9 → x = 7
Проверка: √(7 + 2) = √9 = 3 - верно
Ответ: x = 7

Пример 2: √(2x - 1) = x - 2

ОДЗ: 2x - 1 ≥ 0 → x ≥ 0,5 и x - 2 ≥ 0 → x ≥ 2
Возводим в квадрат: 2x - 1 = (x - 2)² → 2x - 1 = x² - 4x + 4
x² - 6x + 5 = 0 → x₁ = 1, x₂ = 5
Проверка: x = 1 - не входит в ОДЗ
x = 5: √(10 - 1) = 3, 5 - 2 = 3 - верно
Ответ: x = 5

Метод 2: Замена переменной

Используется для уравнений с повторяющимися радикалами

Пример: x + √(x - 2) = 4

Пусть t = √(x - 2), тогда t ≥ 0 и x = t² + 2
Подставляем: t² + 2 + t = 4 → t² + t - 2 = 0
t₁ = 1, t₂ = -2 (не подходит, т.к. t ≥ 0)
√(x - 2) = 1 → x - 2 = 1 → x = 3
Проверка: 3 + √(3 - 2) = 3 + 1 = 4 - верно
Ответ: x = 3

Метод 3: Уединение радикала и возведение в степень

Для уравнений с несколькими радикалами

Пример: √(x + 3) + √(x - 2) = 5

ОДЗ: x + 3 ≥ 0 → x ≥ -3 и x - 2 ≥ 0 → x ≥ 2, значит x ≥ 2
Уединяем один радикал: √(x + 3) = 5 - √(x - 2)
Возводим в квадрат: x + 3 = 25 - 10√(x - 2) + x - 2
Упрощаем: 10√(x - 2) = 20 → √(x - 2) = 2
Возводим в квадрат: x - 2 = 4 → x = 6
Проверка: √(6 + 3) + √(6 - 2) = 3 + 2 = 5 - верно
Ответ: x = 6

Иррациональные неравенства:
Основные типы и методы решения:

Тип 1: √[n]{f(x)} < g(x)

Для четных n решаем систему:

  • f(x) ≥ 0 (ОДЗ)
  • g(x) > 0
  • f(x) < [g(x)]ⁿ

Пример: √(x - 1) < 3

Система: x - 1 ≥ 0 → x ≥ 1
3 > 0 - верно
x - 1 < 9 → x < 10
Ответ: x ∈ [1; 10)

Тип 2: √[n]{f(x)} > g(x)

Для четных n решаем совокупность двух систем:

Система 1: g(x) < 0 и f(x) ≥ 0

Система 2: g(x) ≥ 0 и f(x) > [g(x)]ⁿ

Пример: √(x + 2) > x

Система 1: x < 0 и x + 2 ≥ 0 → x ≥ -2
Решение: x ∈ [-2; 0)

Система 2: x ≥ 0 и x + 2 > x²
x² - x - 2 < 0 → (x - 2)(x + 1) < 0 → x ∈ (-1; 2)
С учетом x ≥ 0: x ∈ [0; 2)

Объединяем: x ∈ [-2; 0) ∪ [0; 2) = [-2; 2)
Ответ: x ∈ [-2; 2)

Особые случаи и замечания:
Тип уравнения/неравенства Особенности решения Пример
Корень четной степени Обязательно ОДЗ: подкоренное выражение ≥ 0 √(x - 3), ОДЗ: x ≥ 3
Корень нечетной степени ОДЗ не требуется, определен для всех x √[3](x + 1) определен при всех x
Несколько радикалов ОДЗ - пересечение ОДЗ всех радикалов √(x - 1) + √(3 - x), ОДЗ: x ∈ [1; 3]
Посторонние корни Всегда проверять полученные решения При возведении в квадрат могут появиться лишние корни
Типичные ошибки и как их избежать:
  • Не проверять ОДЗ - всегда находите область определения
  • Не делать проверку - после возведения в степень обязательно проверяйте корни
  • Неправильно записывать ответ для неравенств - используйте метод интервалов
  • Забывать про знак при возведении в квадрат неравенств
Практические примеры для самопроверки:

Уравнение 1: √(2x + 5) = x + 1

Уравнение 2: √(x + 1) - √(x - 1) = 1

Неравенство 1: √(x² - 4) ≤ x + 2

Неравенство 2: √(3 - x) > x - 1

Алгоритм решения иррациональных уравнений:
  1. Найти ОДЗ (для корней четной степени)
  2. Выбрать метод решения
  3. Выполнить преобразования
  4. Найти корни
  5. Проверить корни в исходном уравнении
  6. Исключить посторонние корни
  7. Записать ответ

Степень с рациональным показателем. Свойства степени. Преобразование выражений, содержащих рациональные степени

Степень с рациональным показателем:
Определение:

Степень с рациональным показателем определяется через корни:

a^{m/n} = √[n]{a^m} = (√[n]{a})^m

где:

  • a - основание степени (a > 0)
  • m/n - рациональный показатель (дробь в несократимом виде)
  • m - целое число
  • n - натуральное число (n ≥ 2)

Примеры:

  • 8^{1/3} = √[3]{8} = 2
  • 16^{3/4} = (√[4]{16})^3 = 2^3 = 8
  • 27^{-2/3} = 1/(27^{2/3}) = 1/((√[3]{27})^2) = 1/9
  • (1/25)^{-1/2} = 25^{1/2} = 5
Особые случаи:

Нулевой показатель: a^0 = 1 (при a ≠ 0)

Отрицательный показатель: a^{-m/n} = 1/(a^{m/n})

Единичный показатель: a^{1/n} = √[n]{a}

Целый показатель: a^{n/1} = a^n

Свойства степеней с рациональными показателями:
Свойство Формула Ограничения Пример
Умножение степеней a^{p} · a^{q} = a^{p+q} a > 0 2^{1/2} · 2^{1/3} = 2^{5/6}
Деление степеней a^{p} ÷ a^{q} = a^{p-q} a > 0 3^{4/3} ÷ 3^{1/3} = 3^{1} = 3
Степень степени (a^{p})^{q} = a^{pq} a > 0 (4^{1/2})^{3} = 4^{3/2} = 8
Степень произведения (ab)^{p} = a^{p}b^{p} a > 0, b > 0 (9·4)^{1/2} = 9^{1/2}·4^{1/2} = 3·2 = 6
Степень частного (a/b)^{p} = a^{p}/b^{p} a > 0, b > 0 (27/8)^{2/3} = 27^{2/3}/8^{2/3} = 9/4
Связь с корнями a^{m/n} = √[n]{a^m} = (√[n]{a})^m a > 0 8^{2/3} = √[3]{8^2} = √[3]{64} = 4
Примеры применения свойств:

Пример 1: Упростить (x^{1/3} · x^{1/6}) ÷ x^{1/2}

x^{1/3 + 1/6 - 1/2} = x^{2/6 + 1/6 - 3/6} = x^0 = 1

Пример 2: Вычислить (16^{3/4} · 4^{-1/2}) ÷ 2^{1/2}

16^{3/4} = (2^4)^{3/4} = 2^3 = 8
4^{-1/2} = (2^2)^{-1/2} = 2^{-1} = 1/2
8 · 1/2 = 4, затем 4 ÷ 2^{1/2} = 4 ÷ √2 = 2√2

Пример 3: Упростить (a^{2/3}b^{-1/2})^6 ÷ (a^{-1}b^{3})

(a^{2/3}b^{-1/2})^6 = a^4b^{-3}
a^4b^{-3} ÷ (a^{-1}b^{3}) = a^{4-(-1)}b^{-3-3} = a^5b^{-6} = a^5/b^6

Преобразование выражений с рациональными степенями:
Переход от корней к степеням:

√[n]{a^m} = a^{m/n}

Примеры:

  • √a = a^{1/2}
  • √[3]{x^2} = x^{2/3}
  • √[5]{a^3b^2} = a^{3/5}b^{2/5}
  • 1/√[4]{x^3} = x^{-3/4}
Приведение к общему показателю:

Аналогично приведению дробей к общему знаменателю

Пример: Упростить x^{1/2} · x^{1/3}

Приводим к общему показателю 6:
x^{1/2} = x^{3/6}, x^{1/3} = x^{2/6}
x^{3/6} · x^{2/6} = x^{5/6}

Пример: Сравнить 2^{1/2} и 3^{1/3}

Приводим к общему показателю 6:
2^{1/2} = 2^{3/6} = (2^3)^{1/6} = 8^{1/6}
3^{1/3} = 3^{2/6} = (3^2)^{1/6} = 9^{1/6}
Так как 8^{1/6} < 9^{1/6}, то 2^{1/2} < 3^{1/3}

Упрощение сложных выражений:

Пример 1: Упростить (a^{1/2} + a^{-1/2})^2 - (a^{1/2} - a^{-1/2})^2

Используем формулу разности квадратов:
= (a^{1/2} + a^{-1/2} + a^{1/2} - a^{-1/2})(a^{1/2} + a^{-1/2} - a^{1/2} + a^{-1/2})
= (2a^{1/2})(2a^{-1/2}) = 4a^{1/2 - 1/2} = 4a^0 = 4

Пример 2: Упростить (x^{1/3} - y^{1/3})(x^{2/3} + x^{1/3}y^{1/3} + y^{2/3})

Это формула разности кубов:
= (x^{1/3})^3 - (y^{1/3})^3 = x - y

Пример 3: Упростить (a^{1/2}b^{-1/3})^6 ÷ (a^{-2}b^{4})

(a^{1/2}b^{-1/3})^6 = a^3b^{-2}
a^3b^{-2} ÷ (a^{-2}b^{4}) = a^{3-(-2)}b^{-2-4} = a^5b^{-6} = a^5/b^6

Решение уравнений с рациональными степенями:
Методы решения:

Метод 1: Приведение к одинаковому основанию

Пример: 8^{x} = 4

(2^3)^x = 2^2 → 2^{3x} = 2^2 → 3x = 2 → x = 2/3

Метод 2: Замена переменной

Пример: x^{2/3} - 3x^{1/3} + 2 = 0

Пусть t = x^{1/3}, тогда t^2 - 3t + 2 = 0
t_1 = 1, t_2 = 2
x^{1/3} = 1 → x = 1, x^{1/3} = 2 → x = 8

Практическое применение:

В физике: Формулы с дробными показателями для описания нелинейных процессов

В геометрии: Длина диагонали куба: d = a√3 = a·3^{1/2}

В финансах: Сложные проценты с дробными периодами

В биологии: Модели роста популяций

Важные замечания:
  • Основание степени должно быть положительным: a > 0
  • Дробь в показателе должна быть несократимой
  • При преобразованиях следите за ОДЗ
  • Свойства степеней работают только для положительных оснований
  • Рациональные степени удобны для сравнения чисел и упрощения выражений

Показательные уравнения

Определение и основные понятия:
Определение:

Показательное уравнение - уравнение, в котором неизвестная переменная содержится в показателе степени.

Общий вид: a^{f(x)} = b^{g(x)} или a^{f(x)} = c

где a > 0, a ≠ 1, b > 0, b ≠ 1, c > 0

Примеры показательных уравнений:

  • 2^x = 8
  • 3^{x+1} = 27
  • 5^{2x} - 6·5^x + 5 = 0
  • 2^{x+1} + 2^{x-1} = 5
Основные методы решения показательных уравнений:
Метод 1: Приведение к одинаковому основанию

Если a^{f(x)} = a^{g(x)}, то f(x) = g(x)

Пример 1: 2^x = 8

2^x = 2^3 → x = 3

Пример 2: 9^{x-1} = 27^{2x}

(3^2)^{x-1} = (3^3)^{2x} → 3^{2(x-1)} = 3^{6x}
2(x - 1) = 6x → 2x - 2 = 6x → -4x = 2 → x = -1/2

Пример 3: 4^x + 2^{x+1} = 80

(2^2)^x + 2·2^x - 80 = 0 → (2^x)^2 + 2·2^x - 80 = 0
Пусть t = 2^x > 0, тогда t^2 + 2t - 80 = 0
t_1 = 8, t_2 = -10 (не подходит)
2^x = 8 → 2^x = 2^3 → x = 3

Метод 2: Вынесение общего множителя

Используется, когда степени имеют одинаковое основание с разными показателями

Пример 1: 2^{x+1} + 2^{x-1} = 5

2^x·2 + 2^x/2 = 5 → 2^x(2 + 1/2) = 5 → 2^x·(5/2) = 5
2^x = 5 · (2/5) = 2 → 2^x = 2^1 → x = 1

Пример 2: 3^{x+2} - 3^{x-1} = 26

3^x·9 - 3^x/3 = 26 → 3^x(9 - 1/3) = 26 → 3^x·(26/3) = 26
3^x = 26 · (3/26) = 3 → 3^x = 3^1 → x = 1

Пример 3: 5^{2x-1} - 5^{2x-3} = 4,8

5^{2x}/5 - 5^{2x}/125 = 4,8 → 5^{2x}(1/5 - 1/125) = 4,8
5^{2x}·(24/125) = 4,8 → 5^{2x} = 4,8·125/24 = 25
5^{2x} = 5^2 → 2x = 2 → x = 1

Метод 3: Замена переменной

Используется для уравнений вида a^{2x} + p·a^x + q = 0

Пример 1: 4^x - 3·2^x + 2 = 0

(2^x)^2 - 3·2^x + 2 = 0
Пусть t = 2^x > 0, тогда t^2 - 3t + 2 = 0
t_1 = 1, t_2 = 2
2^x = 1 → x = 0, 2^x = 2 → x = 1
Ответ: x = 0, x = 1

Пример 2: 9^x - 4·3^x + 3 = 0

(3^x)^2 - 4·3^x + 3 = 0
Пусть t = 3^x > 0, тогда t^2 - 4t + 3 = 0
t_1 = 1, t_2 = 3
3^x = 1 → x = 0, 3^x = 3 → x = 1

Пример 3: 2^{2x} + 2^x - 12 = 0

Пусть t = 2^x > 0, тогда t^2 + t - 12 = 0
t_1 = 3, t_2 = -4 (не подходит)
2^x = 3 → x = log₂3

Метод 4: Логарифмирование

Используется, когда нельзя привести к одинаковому основанию

Пример 1: 2^x = 5

Логарифмируем обе части: log(2^x) = log5 → x·log2 = log5
x = log5 / log2 = log₂5

Пример 2: 3^{x+1} = 7^{x-1}

Логарифмируем: (x+1)log3 = (x-1)log7
x·log3 + log3 = x·log7 - log7
x(log3 - log7) = -log7 - log3
x = -(log7 + log3)/(log3 - log7) = (log7 + log3)/(log7 - log3)

Пример 3: 5^{2x} = 3^{x+1}

Логарифмируем: 2x·log5 = (x+1)log3
2x·log5 = x·log3 + log3
x(2log5 - log3) = log3
x = log3/(2log5 - log3)

Специальные типы показательных уравнений:
Тип уравнения Метод решения Пример
Однородные Деление на одну из степеней 4^x - 2^x = 2^{x+1}
Квадратные относительно показательной функции Замена переменной 9^x - 8·3^x - 9 = 0
С разными основаниями Логарифмирование 2^x = 3^{x-1}
С параметром Исследование в зависимости от параметра a·2^x - 4 = 0
Решение систем показательных уравнений:

Пример 1:

{ 2^x · 3^y = 72; 2^y · 3^x = 108 }

Перемножим уравнения: 2^{x+y} · 3^{x+y} = 7776 → 6^{x+y} = 6^5 → x + y = 5
Поделим первое уравнение на второе: 2^{x-y} · 3^{y-x} = 2/3 → (2/3)^{x-y} = (2/3)^1 → x - y = 1
Решаем систему: x + y = 5, x - y = 1 → x = 3, y = 2

Пример 2:

Перемножим: 10^{x+y} = 1600 → x + y = log₁₀1600 = 2 + log₁₀16

Решаем систему относительно x и y

Показательные неравенства:

Основное правило: При a > 1: a^{f(x)} > a^{g(x)} ⇔ f(x) > g(x)
При 0 < a < 1: a^{f(x)} > a^{g(x)} ⇔ f(x) < g(x)

Пример 1: 2^{x-1} > 8

2^{x-1} > 2^3 → x - 1 > 3 → x > 4 (т.к. 2 > 1)

Пример 2: (1/3)^{x+2} < 9

(1/3)^{x+2} < (1/3)^{-2} → x + 2> -2 → x > -4 (т.к. 0 < 1/3 < 1, знак меняется)

  2. Выбрать подходящий метод решения
  3. Выполнить преобразования
  4. Найти корни
  5. Сделать проверку (если необходимо)
  6. Записать ответ

Важные замечания:
  • Показательная функция всегда положительна: a^x > 0 для всех x
  • При решении проверяйте ОДЗ
  • При замене переменной не забывайте про условие t > 0
  • Внимательно следите за знаком при решении неравенств

Логарифм числа. Десятичные логарифмы. Преобразование выражений, содержащих логарифмы

Определение логарифма:
Основное определение:

Логарифм числа b по основанию a - это показатель степени, в которую нужно возвести основание a, чтобы получить число b.

logₐb = c ⇔ a^c = b

где:

  • a - основание логарифма (a > 0, a ≠ 1)
  • b - логарифмируемое число (b > 0)
  • c - значение логарифма

Примеры:

  • log₂8 = 3, так как 2^3 = 8
  • log₅1 = 0, так как 5^0 = 1
  • log₃(1/9) = -2, так как 3^{-2} = 1/9
  • log₇7 = 1, так как 7^1 = 7
Основное логарифмическое тождество:

a^{logₐb} = b

Примеры:

  • 2^{log₂5} = 5
  • 10^{lg100} = 100
  • e^{ln3} = 3
Виды логарифмов:
Десятичные логарифмы:

Десятичный логарифм - логарифм по основанию 10.

Обозначение: lg b = log₁₀b

Примеры:

  • lg 10 = 1
  • lg 100 = 2
  • lg 1000 = 3
  • lg 0,1 = -1
  • lg 0,01 = -2
Натуральные логарифмы:

Натуральный логарифм - логарифм по основанию e (e ≈ 2,71828...)

Обозначение: ln b = logₑb

Примеры:

  • ln e = 1
  • ln e² = 2
  • ln 1 = 0
Другие специальные логарифмы:

Двоичный логарифм: log₂b (в информатике)

Примеры:

  • log₂2 = 1
  • log₂4 = 2
  • log₂8 = 3

Упрощение сложных выражений:

Пример 1: Упростить (lg5 + lg2) · lg10

(lg5 + lg2) · lg10 = lg(5·2) · 1 = lg10 · 1 = 1

Логарифмические уравнения

Определение и основные понятия:
Определение:

Логарифмическое уравнение - уравнение, в котором неизвестная переменная содержится под знаком логарифма или в основании логарифма.

Общий вид: logₐf(x) = b или logₐf(x) = logₐg(x)

где a > 0, a ≠ 1, f(x) > 0, g(x) > 0

Примеры логарифмических уравнений:

  • log₂(x + 1) = 3
  • log₃(x² - 1) = log₃(2x + 1)
  • lg(x - 2) + lg(x + 2) = lg5
  • logₓ(2x - 1) = 2
Основные методы решения логарифмических уравнений:
Метод 1: Потенцирование

Если logₐf(x) = logₐg(x), то f(x) = g(x)

Пример 1: log₃(x² - 1) = log₃(2x + 1)

ОДЗ: x² - 1 > 0 и 2x + 1 > 0
Потенцируем: x² - 1 = 2x + 1
x² - 2x - 2 = 0
x₁ = 1 + √3, x₂ = 1 - √3
Проверяем ОДЗ: x₂ = 1 - √3 ≈ -0,73 не удовлетворяет 2x + 1 > 0
Ответ: x = 1 + √3

Пример 2: lg(x - 2) + lg(x + 2) = lg5

ОДЗ: x - 2 > 0 и x + 2 > 0 → x > 2
lg[(x - 2)(x + 2)] = lg5
lg(x² - 4) = lg5
Потенцируем: x² - 4 = 5 → x² = 9 → x = ±3
С учетом ОДЗ: x = 3
Ответ: x = 3

Метод 2: Приведение к одному основанию

Используется, когда логарифмы имеют разные основания

Пример 1: log₂x + log₄x = 3

ОДЗ: x > 0
Приводим к основанию 2: log₄x = log₂x / log₂4 = (1/2)log₂x
log₂x + (1/2)log₂x = 3 → (3/2)log₂x = 3 → log₂x = 2 → x = 4
Проверка ОДЗ: x = 4 > 0 - верно
Ответ: x = 4

Пример 2: log₃x + log₉x + log₈₁x = 7

ОДЗ: x > 0
Приводим к основанию 3:
log₉x = log₃x / log₃9 = (1/2)log₃x
log₈₁x = log₃x / log₃81 = (1/4)log₃x
log₃x + (1/2)log₃x + (1/4)log₃x = 7
(7/4)log₃x = 7 → log₃x = 4 → x = 3⁴ = 81
Ответ: x = 81

Метод 3: Замена переменной

Используется для уравнений вида A·log²ₐf(x) + B·logₐf(x) + C = 0

Пример 1: log²₂x - 3log₂x + 2 = 0

ОДЗ: x > 0
Пусть t = log₂x, тогда t² - 3t + 2 = 0
t₁ = 1, t₂ = 2
log₂x = 1 → x = 2, log₂x = 2 → x = 4
Оба корня удовлетворяют ОДЗ
Ответ: x = 2, x = 4

Пример 2: lg²x - lg x² - 3 = 0

ОДЗ: x > 0
lg x² = 2lg x
lg²x - 2lg x - 3 = 0
Пусть t = lg x, тогда t² - 2t - 3 = 0
t₁ = 3, t₂ = -1
lg x = 3 → x = 1000, lg x = -1 → x = 0,1
Ответ: x = 1000, x = 0,1

Метод 4: Использование определения логарифма

Если logₐf(x) = b, то f(x) = aᵇ

Пример 1: log₂(x + 1) = 3

ОДЗ: x + 1 > 0 → x > -1
По определению: x + 1 = 2³ = 8 → x = 7
Проверка ОДЗ: 7 > -1 - верно
Ответ: x = 7

Пример 2: logₓ(2x - 1) = 2

ОДЗ: x > 0, x ≠ 1, 2x - 1 > 0 → x > 1/2
По определению: 2x - 1 = x²
x² - 2x + 1 = 0 → (x - 1)² = 0 → x = 1
Но x ≠ 1 по ОДЗ ⇒ решений нет
Ответ: корней нет

Тип уравнения Метод решения Пример
С переменным основанием Учет ОДЗ и проверка logₓ(2x - 1) = 2
С модулем Раскрытие модуля log₂|x - 1| = 1
С параметром Исследование в зависимости от параметра logₐ(x - 1) = 2
Системы уравнений Комбинация методов { log₂x + log₂y = 3; x + y = 6 }
Решение систем логарифмических уравнений:

Пример 1:

{ log₂x + log₂y = 3; x + y = 6 }

ОДЗ: x > 0, y > 0
log₂x + log₂y = log₂(xy) = 3 → xy = 2³ = 8
Решаем систему: { x + y = 6; xy = 8 }
По теореме Виета: x и y - корни уравнения t² - 6t + 8 = 0
t₁ = 2, t₂ = 4
Ответ: (2; 4), (4; 2)

Пример 2:

{ lg x + lg y = 1; x² + y² = 13 }

ОДЗ: x > 0, y > 0
lg x + lg y = lg(xy) = 1 → xy = 10
(x + y)² = x² + 2xy + y² = 13 + 20 = 33 → x + y = √33
Решаем систему: { x + y = √33; xy = 10 }
По теореме Виета: x и y - корни уравнения t² - √33 t + 10 = 0

Логарифмические неравенства:

Основное правило:
При a > 1: logₐf(x) > logₐg(x) ⇔ f(x) > g(x) > 0
При 0 < a < 1: logₐf(x) > logₐg(x) ⇔ 0 < f(x) < g(x)

Пример 1: log₂(x - 1) > 3

ОДЗ: x - 1 > 0 → x > 1
log₂(x - 1) > log₂8 → x - 1 > 8 → x > 9 (т.к. 2 > 1)
С учетом ОДЗ: x > 9

Пример 2: log_{1/2}(x + 1) > log_{1/2}(2x - 1)

ОДЗ: x + 1 > 0 и 2x - 1 > 0 → x > 1/2
Так как 0 < 1/2 < 1, знак меняется:
x + 1 < 2x - 1 → -x < -2 → x> 2
С учетом ОДЗ: x > 2

  3. Выполнить преобразования
  4. Найти корни
  5. Проверить корни на принадлежность ОДЗ
  6. Исключить посторонние корни
  7. Записать ответ
